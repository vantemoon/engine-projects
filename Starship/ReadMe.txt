Starship Playable (SD1-A03)


Build Instructions:

Open Starship.sln in Visual Studio.
Build the solution in Debug or Release mode for x64.
The executable will be placed in SD/Starship/Run as Starship_[Debug|Release]_x64.exe.


How to Play:

The game starts in Attract mode. Press Space, N, or a controller button (A or Start) to begin; Escape exits or returns to Attract mode after a game ends.


Controls:

Keyboard
S / F: Rotate ship left / right
E: Apply thrust
Space: Fire a bullet
N: Respawn if dead
I: Spawn a new Asteroid
F1: Toggle debug mode
F8: Hard-reset the game
T: Hold to slow time to 1/10th speed
P: Pause or unpause
O: Advance one frame, then pause
Esc: Return to Attract mode or exit if already there
K: Kill all hostile entities (developer trick, skip to next wave)

Xbox Controller
Left joystick: Facing and thrust
A or Right Trigger: Fire a bullet
Start: Respawn if dead
Back: Return to Attract mode or exit if already there


Gameplay Details:

The PlayerShip can be controlled by keyboard or Xbox controller. On keyboard, use S/F to rotate, E to thrust, Space to fire, and N to respawn. On controller, the left joystick sets facing and thrust, A or the right trigger fires, and Start respawns.

Enemies include Beetles, which move at a fixed speed, and Wasps, which accelerate. Both face the player, damage on touch, and take multiple hits to destroy. Asteroids wrap around screen edges, and Debris appears on deaths but fades out quickly.

Gameplay progresses in waves, each harder than the last, with victory after the fifth wave. The player has four lives, shown as ship icons; once all lives are lost, the game ends and returns to Attract mode.
