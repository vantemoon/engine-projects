\# Custom Engine Projects



A collection of game projects and technical demos built using my custom C++ game engine.



This repository includes coursework and personal projects focused on gameplay programming, rendering, engine systems, and real-time interaction. Each project was developed to explore different areas of game development, including input handling, camera control, physics, UI, animation, and 2D/3D rendering.



\## Projects



Each folder contains a separate project built on top of the custom engine. Project-specific details, controls, and features may be included in their own README files.



\## Technologies



\- C++

\- Custom game engine

\- Real-time rendering

\- Gameplay systems

\- Developer console tools



\## Purpose



This repository is intended to showcase my game programming work, engine-based development experience, and ability to build interactive systems from the ground up.
